# ☕ Bootcamp Java Backend com IA

Bem-vindo ao meu repositório de estudos do **Bootcamp Java Backend com Inteligência Artificial**.

Este espaço foi criado para documentar minha jornada de aprendizado, armazenando exercícios, desafios de código, projetos práticos e anotações desenvolvidos ao longo do bootcamp.

## 📖 Sobre o Bootcamp

O bootcamp tem como objetivo capacitar desenvolvedores na construção de aplicações backend robustas, seguras e escaláveis utilizando **Java 21** e o ecossistema **Spring**.

Além dos fundamentos do desenvolvimento backend, o programa aborda conceitos modernos de arquitetura de software e integração com ferramentas de Inteligência Artificial Generativa.

## 🚀 Principais Tecnologias e Conceitos

* Java 21
* Programação Orientada a Objetos (POO)
* Estruturas de Controle e Coleções
* APIs REST
* Spring Boot
* Spring Security

* OpenAI Whisper
* Google Gemini
* Inteligência Artificial Generativa
